Quick Start (Canva)
  1. Upload your asset (or the demo PDF).
  2. Place buyer-ID text @ 35% opacity:
    {buyer_name} · {order_id} · {date} · {proof_id}
  3. (Optional) Client logo:
    • Faint tile: tiny logo repeated, 8–12% opacity, behind content.
    • Corner tag: small logo top-right, 60–70% opacity, then Lock.
  4. Apps → Bulk Create → Upload CSV → map fields → Generate multiple designs.
  5. Download → ZIP (PDF/PNG). Deliver + include DMCA template.

Best practice: don’t put the TruthLayer logo on client deliverables by default. Use buyer-ID text always; client logo only if requested.
